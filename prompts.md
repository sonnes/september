## Home

Create a 3-column layout with the following sections:

1. Navigation
2. List of conversations
3. Selected Conversations

the layout should be responsive and should look good on all devices.

load the conversations from the `/api/conversations/list` endpoint using react-query. implement the api by using mock data from the `data/conversations.json` file.

on selecting a conversation from the list, display the conversation in the selected conversation section. fetch the conversation details from the `/api/conversations/:id` endpoint using react-query. implement the api by using mock data from the `data/conversation.json` file.
